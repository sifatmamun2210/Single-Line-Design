# -Single-Line-Design-
110 kV Power System Modelling and Single Line Design using DIgSILENT PowerFactory.
Developed a 110 kV transmission network model using DIgSILENT PowerFactory. 
Drafted high-voltage single line diagrams (SLD) for power system representation. 
Performed power network modelling and component parameter configuration. 
Applied electrical grid design principles including transformer HV–LV connections. 
Utilised simulation tools for power system analysis and network visualisation.
